


# comandos a utilizar:  
- docker compose up:  Necesario para iniciar /levantar el sistema. 
- docker compose down: Apaga el sistema. 

# ¿Cómo acceder y utilizar la aplicación? 
 - Una vez se haya iniciado/levantado el sistema con docker acceder a: localhost:8080. 
 - De esta manera la app se iniciará en pantalla.
